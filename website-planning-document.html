Draconia Atiesh - Website Planning Document

Site Name & Logo

Site Name: Draconia Atiesh

This name combines the imagery of "Draconia" (referencing dragons, a core fantasy element in Warcraft) with "Atiesh" (the name of the server). The name aims to be memorable and immediately relevant to the guild's purpose and game context.

Logo Idea: A stylized, gold dragon head with glowing red eyes, against a dark blue background.

Site Purpose

The site serves as a central hub for the "Draconia" guild on the "Atiesh" realm in Mists of Pandaria Classic. Its primary functions are to foster guild camaraderie, provide a platform for members to sign up for scheduled raids, and offer a simple application process for new members to join.

Scenarios

Below are two common scenarios that a site visitor might have, which the website's content and design will address:

  1. How do I join the guild?

    A new visitor to the site will be able to navigate to a dedicated "Join" or "Apply" page. This page will contain a simple form for them to fill out their character information, class, and experience. Once submitted, the form will email the guild leaders, notifying them of a new application.

  2. How do I sign up for raids?

    A guild member visiting the site will go to the "Raid Signups" page. This page will display a calendar or list of upcoming raids with dates and times. Each raid event will have a button or link that allows the member to sign up and indicate their availability and role (e.g., Tank, Healer, DPS).

Color Schema

The chosen color schema is inspired by a classic fantasy aesthetic and the guild's name.

  • Blood Red (#880808): Used for accents, borders, and main calls-to-action to grab attention. This color will be used for headings (H2s) and key UI elements.
  • Blue (#00224b): The primary background color for the site, providing a deep, serious, and professional tone.
  • Gold (#bb8010): Used for important text like the main title (H1) and as a border color, symbolizing prestige and achievement.

Typography

The font choices balance readability with a thematic fantasy feel.

  • Headings (H1, H2): 'Playfair Display', serif;

    Example: This Is a Guild Title

    A bold, classic serif font that gives the headings a strong, established look.

  • Body Text (Paragraphs, Lists): 'Roboto', sans-serif;

    Example: The guild website provides a hub for members.

    A clean, modern, and highly readable sans-serif font for all body content, ensuring text is easy to read at any size.
